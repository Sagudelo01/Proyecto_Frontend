## 📋 Descripción

Interfaz de e-commerce donde los usuarios pueden explorar, filtrar y comprar ropa desde cualquier dispositivo. Incluye secciones de abrigos, camisetas y pantalones con precios, carrito de compras y descripciones detalladas por producto.
